# SmartConfAPI

SmartConf REST API created as part of OOP laboratory.

## API documentation
ConferenceController
* **GET** /conferences
* **GET** /conferences/{id}
* **GET** /conferences?name
* **GET** /conferences?nameStartingWith
* **GET** /conferences?catName
* **POST** /conferences
* **PUT** /conferences/{id}
* **DELETE** /conferences/{id}

CategoryController
* **GET** /categories
